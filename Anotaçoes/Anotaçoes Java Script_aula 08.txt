Operadores do Java Script: 
1 - Aritimeticos (Matematica)
2-  Atribuição 
3 - Relacionais 
4 - Logicos 
5 - ternario 


Ordem de precedencia aritimetica

1 - ()
2 - **
3 - * / %
4 - + e - 

Operadores Relacionais
> maior que
< menor que 
>= maior ou igual a 
<= menor ou igual a 
== igual
=== identico
!= Diferente de  

Logicos: 
! Negação
&& Conjunção (E) (And)
|| Disjunção (OU) (Or)

Precedencia: 
1 - () ** / 
2 - > < >= 
3 - ! 
4 - && 
5 - || 


Operador Ternario
?  Oque acontece quando é verdadeiro
:  oque acontece quando é falso 

Expl: media >= 7.0 ?  "Aprovado" //True  : "Reprovado" //false
