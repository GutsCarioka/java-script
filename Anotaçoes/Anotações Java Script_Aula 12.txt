Condiçoes

Estrutura Base de uma Condição IF: 

if(Condição){
    True
}else{
    False
}


Condiçoes Aninhadas: 
    Condiçoes que tem Condiçoes dentro
    Expl:

    if(Condição){
        True
        if(Condiçoes){
            True
        }
    }else{
       if(Condiçoes){
            True
        }else{

        }
    }

    Ou 
    if(Condição){
        True
        if(Condiçoes){
            True
        }
    }else if(Condiçoes){
            True
        }else{

    }
